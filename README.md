For fast verification:

    1. Install dependencies: `pip install -r requirements.txt`

    2. Download images: `python download_unido_images.py`

    3. Train model: `python train_all_specialists.py`

    4. Download checkpoint (optional/ if you already started training, you have to delete the checkpoint for this script to work): `python download_checkpoint.py`

    5. Generate submission: `python submit.py`
